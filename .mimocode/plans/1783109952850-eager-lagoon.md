# Item Report — AG-Grid + Shadcn Implementation Plan

## Overview
Build a comprehensive Item Report page at `/erp/reports/items` with:
- AG-Grid data table with filtering, sorting, column pinning, CSV export
- KPI summary cards (total items, categories, low stock, out of stock, avg price, inventory value)
- Charts (inventory by category, stock by warehouse)
- Filter sidebar (category, type, stock status, price range, warehouse)
- Quick filter chips (All, Saleable, Purchasable, Low Stock, Out of Stock)
- Detail drawer (Sheet) for row click
- Search with suggestions

## Files to Modify/Create

### 1. Backend: New tRPC Procedure
**File:** `server/items/items.router.ts`

Add `report` procedure (query, orgProcedure) that returns ALL items (no pagination) with full data needed for the report:

```
Input: none (or optional minimal filters)
Select: id, sku, barcode, name, image, type, unit, isActive, isSaleable, isPurchasable,
        purchasePrice, salesPrice, averageCost, minStock, reorderPoint, reorderQty,
        weightKg, description, createdAt,
        category: { id, name },
        family: { id, name },
        class: { id, name },
        commodity: { id, name },
        taxRate: { id, name, rate },
        stock: { quantity, warehouse: { id, name } },
        supplierItems: { supplierSku, basePrice, supplier: { id, name } }
```

Post-process to add:
- `totalStock` — sum of stock quantities across warehouses
- `stockByWarehouse` — array of `{ warehouseId, warehouseName, quantity }`
- `stockStatus` — "out" | "low" | "in_stock" (computed from totalStock vs minStock)
- `inventoryValue` — totalStock * purchasePrice

Guard: `assertCan(ctx.ability, 'report:inventory', 'all')` (same as `topItems`)

**Components used:**
- `AgGridReact` + `useTableTheme()` (existing pattern)
- `Card`, `CardContent`, `CardHeader`, `CardTitle` — KPI cards
- `Badge` — status badges
- `Button` — actions
- `Tabs` — not needed, single view
- `Sheet`, `SheetContent`, `SheetHeader`, `SheetTitle` — detail drawer
- `Input`, `InputGroup` — search
- `ScrollArea` — filter sidebar
- Recharts `BarChart`, `PieChart` via `ChartContainer` — charts

**AG-Grid Column Definitions:**
| Col | Field | Width | Notes |
|-----|-------|-------|-------|
| Image | image | 50 | Cell renderer: Avatar or icon fallback |
| SKU | sku | 100 | Monospace font |
| Barcode | barcode | 100 | Monospace font |
| Name | name | 200 | Bold, primary text |
| Category | categoryName | 120 | Badge with category color |
| Family | familyName | 100 | |
| Class | className | 100 | |
| Purchase Price | purchasePrice | 100 | NumberFormatter |
| Sales Price | salesPrice | 100 | NumberFormatter |
| Stock | totalStock | 80 | Color-coded (red/yellow/green) |
| Warehouse | stockWarehouseNames | 150 | Comma-joined warehouse names |
| Status | stockStatus | 100 | Badge: In Stock (green), Low (yellow), Out (red) |
| Type | type | 100 | Badge |
| Actions | — | 60 | Eye icon → opens Sheet |

**AG-Grid Features (Community edition):**
- Sorting, filtering (set filter, text filter)
- Column pinning, resizing, reordering
- Row selection
- CSV export via `exportDataAsCsv()`
- Client-side model (all data loaded at once)

**Filter Sidebar (custom, not AG-Grid filters):**
- Category (checkbox list with counts)
- Type (PRODUCT / SERVICE / BUNDLE)
- Stock Status (In Stock / Low / Out)
- Price Range (min/max inputs)
- Warehouse (checkbox list with counts)
- "Clear All" button

**Quick Filters:**
- All, Saleable, Purchasable, Low Stock, Out of Stock, Recently Added (last 7 days)

**Filter Chips:**
- Shows active filters as dismissible chips
- "Clear all" button

**Detail Drawer (Sheet):**
- Opens on row click (eye icon or double-click)
- Shows full item details: image, SKU, barcode, name, description, category hierarchy, prices, stock per warehouse, suppliers, dimensions
